# Zero — Vision Board FTUX prototype

A standalone, self-contained prototype of Zero's first-time onboarding: an AI guide (Avery)
has a live conversation and compiles the user's answers into a vision board, roadmap,
curriculum, outcome, and launch.

**Live demo:** see the repo's GitHub Pages URL (Settings → Pages).

Single-file build — `index.html` has all JS, CSS, and fonts embedded, so it runs by just
opening it in a browser. Source lives in the design team's working repo.
